# Ruler
Ruler 满足用户对 身高cm inch 和 体重 kg lb 通过支持选择<br>
1.支持ruler滑动时震动，大刻度震动加强.<br>
2.支持刻度宽度调整.<br>
3.支持刻度颜色，字体调整.<br>
4.支持ruler类型更新.<br>
5.支持选中箭头自定义.<br>
6.支持点击跳转.<br>
7.支持各种骚操作自定义.<br>
